Introduction

Welcome to the Atomist User Guide. The Atomist User Guide contains documentation on all aspects of Atomist that its users, beginners through experts, can use to more effectively use Atomist to improve their processes and deliver value more quickly.

If you are not familiar with Atomist, you may want to start with the Atomist Overview. If you do not have Atomist set up in Slack, you may want to go through the Getting Started guide.

The Atomist User Guide is split into three sections:

  • Rug: Atomist’s enabling technology
  • Integrations: External platforms and tools that Atomist works with
  • Interfaces: How users interact with Atomist

Click on any of the links above to go directly to that section, or click Next at the bottom right of this page to go to the next page in the guide.
